# TaskFlow

TaskFlow is a modern AI-powered task management application that helps users create, organize, and manage tasks through an intuitive interface. The application combines traditional productivity features with AI-assisted task creation and task breakdown to improve planning and workflow.

## Features

- Create and delete tasks
- AI-powered natural language task creation
- AI-assisted task breakdown
- Drag-and-drop task reordering
- Search, filter, and sort tasks
- Responsive user interface
- Clean light theme

## Tech Stack

| Technology | Purpose |
|------------|---------|
| HTML5 | Application structure |
| CSS3 | Styling and layout |
| JavaScript (ES6) | Application logic |
| Bootstrap 5 | Responsive UI components |
| SortableJS | Drag-and-drop functionality |
| Anthropic Claude API | AI-powered task generation |

Clone the repository:

```bash

```

Navigate to the project directory:

```bash

```
